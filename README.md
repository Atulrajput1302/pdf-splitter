# pdf-splitter
Splits the pdf on the basis of provided index

# Working:
* Upload your pdf and provide the starting page and ending page index.
* Then extract the new splitted pdf
* Finally, download it




### Modules Used
* Flask  --> for the website
* from PyPDF2 we have import 
    * PdfFileReader
    * PdfFileWriter
